# TP1_EngSoftware
**MEMBROS**: Bruno Lopes Melo Fonseca (Full stack), Gabriel Alkmim Barros (Full stack), Felipe Araújo Melo (Full stack)

**SOBRE:**  Este sistema funciona como uma rede social voltada para a boemia de Belo Horizonte, permitindo que usuários cataloguem e avaliem os bares visitados na capital do boteco baseado no site letterboxd. A plataforma conecta amigos através do "check-in" em novos estabelecimentos, criando um diário visual e gastronômico da cidade. Além de registrar experiências, o site permite usuarios descobrir novos lugares, gerando novas experiencias na vida noturna de BH

**TECNOLOGIAS:** python, SQL, flask

**AGENTES:** GPT, Gemini, Claude, COPILOT
